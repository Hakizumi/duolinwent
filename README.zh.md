## 📝 项目介绍
> [!IMPORTANT]
> - 此项目仅供学习,用户需要在**合法范围内**内使用此软件.
> - 此项目仅能在**安卓系统**的手机或模拟器内使用.

## 🚀 快速开始
### 下载已修补的`多邻国`软件
如果你是新手或从未接触过**LSPosed**,推荐此方法

下载最新的`duolingo_patched.apk`并在你的**安卓**手机上安装.
然后您就可以正常使用已修补的**多邻国**,与官方软件完全相同的功能.

### 自己使用`LSPosed`框架来修补`多邻国`
下载最新的`duolinwent.apk`,
然后**LSPosed**框架将会扫描到`duolinwent`并注册为一个**LSPosed**模块,
然后在**LSPosed**中使用`duolinwent`来修补**多邻国**app安装包.
