## 🚀 Quick Start
### Download patched `Duolingo` software
Recommended for newbies or people who have never been exposed to the **LSPosed**.

Download latest `duolingo_patched.apk` and install it on your **Android** mobile phone.
Then you can use the patched **Duolingo** normally as official software with the same features. 

### Patch the `Duolingo` yourself with `LSPosed` framework
Download latest `duolinwent.apk`,
then the **LSPosed** framework will scan the `duolinwent` as a **LSPosed** module,
then patch the **Duolingo** app installation package with the `duolinwent` in **LSPosed**.
